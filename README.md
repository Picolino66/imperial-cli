# Imperial-Cliente

Projeto do front-end do sistema Imperial.

## Requisitos
Este projeto foi feito utilizando:

Node: v18.13.0

Npm: 8.19.3

Angular CLI: 10.0.5

## Instalação
Para instalar as dependências:

npm install

## Configuração
No arquivo environment.ts (src/environments/environment.ts)

Está a url para a nossa api.
